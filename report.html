데이터 소스 우선순위

시장 지수/등락/수급 데이터:

  1. MSSQL T_MarketIndicator (USP_MarketIndicator_List_Admin SP)
  2. PG crawler.market_indicator (group_code='KoreaIndexEnd')
  3. PG shared.market_index_day / shared.stock_day (폴백)

특징 테마:

  1. api-radar API (POST /themelog/top-themes)

상한가/하한가/뉴스:

  1. MSSQL Stored Procedure (USP_StockExpressionResult_List_Report, USP_NewsRelation_List_Report)

데이터베이스 쿼리 상세

MSSQL Stored Procedures

  • USP_MarketIndicator_List_Admin - 마켓 지표 조회 (group_code, start_date, end_date)
  • USP_StockExpressionResult_List_Report - 상한가/하한가 종목 조회 (date, GroupIdx=2)
  • USP_NewsRelation_List_Report - 특징 뉴스 조회 (date)

PostgreSQL Tables

  • shared.market_index_day - 시장 지수 일별 데이터 (index_code, date, close, diff)
  • shared.stock_day - 종목 일별 데이터 (shcode, date, diff, individual_val, registered_val, institutions_val)
  • crawler.market_indicator - 크롤링 마켓 지표 (date, code, group_code, price)

API Endpoints

  • POST {radarApiUrl}/themelog/top-themes - 테마로그 랭킹 TOP 8 조회
    요청: {"captureIdx":10,"top":8}
    응답: [{"keyword":"테마명","relationStocks":[...]}]
