1. 요약

  • ASP.NET Web API 2 기반 .NET Framework 4.7 API 프로젝트이며 Web API 설정이 중심입니다.
  • 공통 파이프라인은 gzip 요청 해제, CORS, 전역 예외 필터, attribute routing, 기본 API route, API 로그 handler, 상세 오류 반환 정책으로 구성됩니다.
  • 컨트롤러의 명시 Route 기준 93개 URL 항목을 확인했습니다.
  • Redis, MongoDB, SQL Biz 계층, 외부 실시간 StockData API를 혼합 사용합니다.
